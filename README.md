# Big Seven Arcade

Big Seven Arcade is a Python-based casino arcade game that lets players enjoy two classic games with custom twists: Blackjack and Roulette. Players start with 100 tokens and can choose to play games, read the rules, or quit through a simple text-based menu system.

## Features

### Blackjack
- Bet tokens before each round
- Try to reach 21 without going over
- Lucky Seven mechanic (two 7s = instant win)
- Mystery card feature
- Dealer AI system
- Hit or Stay gameplay

### Roulette
- Four-color roulette system
- Random lucky color each round
- Multiple betting options:
  - Number
  - Color
  - Even/Odd
  - Zero
- Different payout multipliers depending on bet type

## Technologies Used
- Python 3
